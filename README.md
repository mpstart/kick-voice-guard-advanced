# Kick Voice Guard - Advanced v2.0

**Mejorada para reducir falsos positivos**

## Instalación
1. Descarga el ZIP o clona este repo
2. En Brave: brave://extensions/ → Cargar extensión sin empaquetar
3. Selecciona la carpeta raíz

## Entrenamiento recomendado
- Usa clips de **10-20 segundos** solamente
- Sube 5-6 muestras diferentes
- Umbral de Confianza: 0.85+
- Sensibilidad: 0.35 o menos
